adar1000¶

class adi.adar1000.adar1000(uri='', beams='BEAM0')¶

Bases: adi.attribute.attribute, adi.context_manager.context_manager

ADAR1000 Beamformer

parameters:
uri: type=string

URI of IIO context with ADAR100(s)

beams: type=string,list[string]

String or list of strings idenifying desired chip select option of ADAR100. This is based on the jumper configuration if the EVAL-ADAR100 is used. These strings are the labels coinciding with each chip select and are typically in the form BEAM0, BEAM1, BEAM2, BEAM3. Use a list when multiple are chips are cascaded together. Dynamic class properties will be created for each beam and signal path.
